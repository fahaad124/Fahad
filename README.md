# Haraj Clone - Frontend Design

This repository contains the frontend design for a Haraj-like website. The project includes HTML, CSS, and JavaScript files to create a basic functionality for the following features:

1. Homepage
2. Login Page
3. Ad Listings
4. Contact/Help Pages

## Project Structure
```
.
├── index.html                # Homepage
├── login.html                # Login page
├── ads.html                  # Ad listings page
├── assets/
│   ├── css/
│   │   ├── style.css         # General styling
│   │   └── ads.css           # Styling specific to ads page
│   ├── js/
│   │   ├── script.js         # Common JavaScript functionality
│   │   └── ads.js            # JavaScript for ads page functionality
│   └── images/               # Images for website assets
└── README.md                 # Project documentation
```

## Features
- Basic homepage listing categories.
- User authentication screen (login page).
- Displaying ads in a grid.
- User-friendly design with responsiveness in mind.
